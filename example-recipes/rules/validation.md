# Validation Rules

## Recipes

- A user cannot have two recipes with the same title
- Recipes must have at least one ingredient and one step
- Tags must be lowercase and contain only letters, numbers, and hyphens
- Cook time and servings must be positive integers

## Reviews

- A user can only leave one review per recipe
- Rating must be a whole number between 1 and 5
- A user cannot review their own recipe

## Favorites

- A user can only favorite a recipe once
- A user cannot favorite their own recipe
